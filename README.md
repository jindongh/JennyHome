# Jenny's Home

# Environment
```
sudo apt-get install python-opencv python-dev
virtualenv env
source env/bin/activate
pip install -r requirements.txt
cp /usr/bin/python2.7/dist-packages/cv* env/lib/python2.7/site-packages
```
# Get Started
```
virtualenv env
source env/bin/activate
./start
```
